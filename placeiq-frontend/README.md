# PlaceIQ Frontend

React + Vite + Tailwind CSS frontend for the PlaceIQ Campus Placement Intelligence Platform.

> **See the main README in the project root for full setup instructions.**

## Quick Start

```bash
# From the project root
cd placeiq-frontend
npm install
npm run dev
```

Runs at: **http://localhost:5173**

Requires the Django backend to be running at **http://localhost:8000**.
